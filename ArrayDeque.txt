In Java, ArrayDeque is a resizable array-based implementation of the Deque interface that allows efficient insertion and removal of elements from both ends.
It can be used as a stack (LIFO) or a queue (FIFO), is faster than LinkedList for most operations, but is not thread-safe


üîπ What is ArrayDeque?
Definition: A double-ended queue (deque) implemented using a resizable array.
Package: java.util
Key Feature: Supports adding/removing elements at both ends in O(1) time.
Declaration Example:
ArrayDeque<Integer> deque = new ArrayDeque<>();

üîπ Core Characteristics
Resizable: Grows dynamically as needed; no fixed capacity limit.
Nulls not allowed: You cannot store null elements.
Not thread-safe: Requires external synchronization for concurrent use.
Performance:
=Faster than LinkedList for queue operations (cache-friendly).
=Faster than Stack when used as a stack



üîπ Common Operations
=====================
Operation	          Method	                                  Description
Add at front	      addFirst(e) / offerFirst(e)	          Inserts element at head
Add at end	          addLast(e) / offerLast(e)               Inserts element at tail
Remove from front	  removeFirst() / pollFirst()	          Removes head element
Remove from end	      removeLast() / pollLast()	              Removes tail element
Peek front	          peekFirst()	                          Retrieves head without removing
Peek end	          peekLast()	                          Retrieves tail without removing

import java.util.ArrayDeque;

public class Demo {
    public static void main(String[] args) {
        ArrayDeque<String> deque = new ArrayDeque<>();

        // Add elements
        deque.addFirst("A");
        deque.addLast("B");
        deque.addLast("C");

        // Remove elements
        System.out.println(deque.removeFirst()); // A
        System.out.println(deque.removeLast());  // C

        // Peek
        System.out.println(deque.peekFirst());   // B
    }
}

Capacity growth: Though dynamic, resizing may cause temporary overhead.
